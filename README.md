# SELS Assistant

Sistema de automação para o APMS.

## Objetivo

Facilitar a geração de demonstrativos e automatizar tarefas repetitivas do APMS.

## Funcionalidades atuais

- Geração de demonstrativos
- Leitura de PDF
- Processamento de Razão
- Processamento de Consignado
- Processamento de NG
